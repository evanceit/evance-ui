## headers prop
The `headers` prop is used to describe what's displayed as columns.

```typescript
{
    readonly key?: string | undefined,
    readonly value?: SelectItemKey<Record<string, any>>,
    readonly title?: string | undefined,
    readonly align?: 'end' | 'start' | 'center' | undefined,
    readonly hidden?: VisibilityRuleProp | undefined,
    readonly width?: string | number | DisplayRuleObject | undefined,
    readonly minWidth?: string | number | DisplayRuleObject | undefined,
    readonly maxWidth?: string | number | DisplayRuleObject | undefined,
    readonly headerProps?: { readonly [prop: string]: any } | undefined,
    readonly cellProps?: { readonly [prop: string]: any } | undefined,
}
```
