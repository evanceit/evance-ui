# EvIcon

Evance's EvIcon component uses SVG-based glyphs designed within a viewBox width and height of `24px`.

<Canvas of={EvIconStories.Primary} />

<Controls />

## SVG glyphs

Evance primarily uses SVG icons.

```html
<script>
import { EvIcon } from "@evance/evance-ui/components";
import { EvanceIcon } from "@evance/evance-ui/icons";
</script>
<template>
    <ev-icon :glyph="EvanceIcon" />
</template>
```

## Webfonts
You can also use webfonts (if installed) using CSS classes.

```html
    <ev-icon class="my-icon" />
```
